## 🎯 使用方式

### 自然语言触发

不需要记任何命令。直接说你的需求：

```
"我昨晚梦见蛇了，这是什么意思？"
"帮我用鲁班尺量一下这个门的尺寸吉凶"
"用易经帮我算一下这笔投资能不能做"
"给我写一首关于春天的七言绝句"
"怎么做正宗的东坡肉？"
"帮我选个搬家的黄道吉日"
"用孙子兵法分析一下我们公司和竞争对手的局势"
"诸葛亮的空城计在现代商业中怎么用？"
"帮我写一副春联"
"这个风水布局有什么问题？"
"给我讲讲明朝灭亡的根本原因"
"用中医理论分析一下我总是失眠怎么办"
"帮我设计一个苏州园林风格的小院子"
"造纸术的原理是什么？能在家复刻吗？"
"鲁班锁的原理和制作方法"
```

### 指定卷宗触发

```

"用军事谋略来..."（第二卷）
"从百工技艺出发..."（第三卷）
```
