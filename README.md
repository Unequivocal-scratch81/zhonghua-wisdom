## 这是什么

一个 AI Skill 文件——给你的 AI 助手装上**中华五千年文明的知识库**。

你的 AI 能帮你写代码、查文档，但它不知道：
- 门框宽度多少算吉利？→ **鲁班尺**
- 总失眠是什么体质问题？→ **黄帝内经·阴阳辨证**
- 都江堰两千年不坏靠什么？→ **鱼嘴分水 + 飞沙堰 + 宝瓶口**
- 怎么用硬币起一卦？→ **易经·三币法**
- 东坡肉的秘诀？→ **"慢着火，少着水，火候足时它自美"**
- 竞争对手太强怎么打？→ **孙子兵法·虚实篇**
- 春联怎么写才合平仄？→ **七言绝句格律 + 意象表**
- 晋商票号凭什么百年不倒？→ **身股制 + 密押 + 号规**

装上这个 Skill，它就知道了。

## 为什么做这个

LLM 的训练语料里有中文，但没有**结构化的中华文明知识库**。

它知道"孔子说过己所不欲勿施于人"，但它不知道：
- 这句话放在法家、道家的框架里怎么对比
- 阴阳平衡的分析步骤是什么
- 鲁班尺上"财病离义官劫害本"八个字怎么看
- 易经六十四卦每一卦的人生指引
- 都江堰鱼嘴分水的物理原理
- 曾国藩家书里"日课四条"具体怎么练

这个 Skill 补的就是这块。
