# 第三卷 · 百工技艺

> 百工者，圣人之作也。——《考工记》
> 
> 四大发明只是冰山一角。中华五千年工匠精神，从青铜铸造到丝绸织造，从鲁班木工到景德镇瓷器，从长城砖石到应县木塔。

## 一、四大发明

### 📜 造纸术

#### 历史脉络
- **西汉**：最早的植物纤维纸（灞桥纸）
- **东汉·蔡伦（105年）**：改进造纸术，用树皮、麻头、破布、渔网为原料
- **唐宋**：宣纸成熟，皮纸、竹纸大发展
- **元明清**：造纸技术传播至世界各地

#### 蔡伦造纸七步法
1. **剉**：切断原料
2. **煮**：蒸煮软化纤维
3. **捣**：舂捣成浆
4. **抄**：用帘抄纸
5. **压**：压去水分
6. **晒**：晾干
7. **揭**：揭下成品

#### 现代复刻方法
**家庭简易造纸**：
- 材料：废纸（报纸、办公纸）、水、搅拌机/擀面杖、纱网框架、毛巾
- 步骤：
  1. 撕碎废纸，泡水24小时
  2. 搅打成浆（搅拌机或手捣）
  3. 倒入大盆加水调稀
  4. 纱网框水平入水，捞起纸浆
  5. 翻扣在毛巾上，压平挤水
  6. 自然晾干
- **技巧**：可加入花瓣、树叶做装饰纸。加入明矾可防洇墨。

#### 宣纸知识
- **原料**：青檀皮 + 沙田稻草
- **特点**：润墨性好，寿命千年，"纸寿千年"
- **分类**：生宣（洇墨适合写意）、熟宣（不洇适合工笔）、半熟宣

### 🧨 火药

#### 发明背景
- 最初是道士炼丹的副产品
- 唐代孙思邈《丹经》记载最早的火药配方
- 配方：硝石（KNO₃）+ 硫磺 + 木炭

#### 配比原理
- **最佳比例**（黑火药）：硝75% + 硫10% + 炭15%
- 硝提供氧，硫降低点燃温度，炭提供燃料
- **安全性**：比例不同会产生不同效果——从烟花到武器

#### 民用应用
- **烟花**：隋唐开始，宋代成熟
- **爆竹**：驱邪迎新
- **采矿开山**：工程爆破

### 🧭 指南针

#### 发展历程
- **战国·司南**：天然磁石磨成勺状，放在地盘上
- **北宋·指南鱼**：薄铁片磁化后浮于水面
- **北宋·指南针**：磁针穿于灯芯浮水，或放于碗沿
- **罗盘**：结合方位盘，航海用

#### 磁化方法
1. **摩擦法**：用磁铁沿同一方向摩擦铁针数十次
2. **磁石吸引法**：用天然磁石长期接触
3. **地磁法**：铁针南北放置，用火烧红后冷却（地磁场磁化）

#### 风水罗盘结构
- **天池**：中央指南针
- **先天八卦**：伏羲八卦方位
- **后天八卦**：文王八卦方位
- **二十四山**：八方各分三，共24个方位
- **六十甲子**：纳音五行
- **应用**：定向、测吉凶、选址

### 🖨️ 印刷术

#### 雕版印刷
- **起源**：隋唐（约7世纪）
- **原理**：在木板上刻出凸起的反字，刷墨后覆纸印刷
- **代表作品**：《金刚经》（868年，现存最早有明确日期的印刷品）
- **活字印刷对比**：雕版适合大量印同一内容，活字适合灵活排版

#### 毕昇活字印刷（1041年）
1. **制字**：用胶泥刻字，每字一印
2. **烧硬**：火烧令坚
3. **排版**：按需排入铁框
4. **印刷**：刷墨覆纸
5. **回收**：用完拆版，泥字可复用

#### 现代体验
- **橡皮章活字**：用橡皮刻字，可反复使用
- **木活字**：浙江东阳仍有传承
- **体验**：了解排版、刷墨、压印的全流程

## 二、鲁班术与木工

### 🪚 鲁班其人
- **生卒**：约前507-前444
- **本名**：公输班（鲁国人，故称鲁班）
- **尊称**：百工圣祖、土木工匠始祖

### 鲁班发明一览

| 发明 | 原理 | 现代传承 |
|------|------|---------|
| **锯子** | 受茅草叶齿启发 | 所有锯类工具的祖先 |
| **墨斗** | 墨线弹出直线 | 木工至今使用 |
| **曲尺（鲁班尺）** | 直角测量+吉凶刻度 | 风水必备工具 |
| **刨子** | 削平木材表面 | 木工基础工具 |
| **钻子** | 旋转切削打孔 | 各种钻的祖先 |
| **云梯** | 攻城用可折叠梯 | 军事工程 |
| **石磨** | 碾磨谷物 | 食品加工 |
| **榫卯** | 木构件互锁连接 | 中国传统建筑核心 |

### 🔧 鲁班尺（门光尺）

#### 用途
- 量度门窗家具尺寸，判断吉凶
- 一尺 = 46.08 cm（传统），每尺分八格

#### 八字吉凶
1. **财**（吉）：财德、宝库、六合、迎福
2. **病**（凶）：退财、公事、牢执、孤寡
3. **离**（凶）：长库、劫财、官鬼、失脱
4. **义**（吉）：添丁、益利、贵子、大吉
5. **官**（吉）：顺科、横财、进益、富贵
6. **劫**（凶）：死别、退口、离乡、财失
7. **害**（凶）：灾至、死绝、病临、口舌
8. **本**（吉）：财至、登科、进宝、兴旺

#### 使用方法
- 量门框宽度、窗框宽度、家具尺寸
- 看落点在哪个字上——落在"财、义、官、本"为吉
- **口诀**：「财病离义官劫害本」循环

#### 现代应用
- 买房装修时量门窗尺寸
- 选择吉利尺寸做家具
- 设计房间布局时参考

### 🔩 榫卯结构

#### 基本原理
不使用钉子和胶水，仅靠木构件的凹凸咬合连接。
- **榫**：凸出部分（公）
- **卯**：凹入部分（母）

#### 经典榫卯类型

| 名称 | 结构 | 用途 |
|------|------|------|
| **燕尾榫** | 梯形互锁 | 抽屉、箱体 |
| **粽角榫** | 三角交接 | 柜子转角 |
| **抱肩榫** | 束腰结构 | 椅子腿和座面 |
| **夹头榫** | 腿夹牙条 | 桌案类 |
| **插肩榫** | 腿嵌牙条 | 桌案类 |
| **霸王枨** | 斜撑加固 | 桌子底部 |
| **楔钉榫** | 圆弧连接 | 圈椅扶手 |
| **走马销** | 可拆卸连接 | 需要组装的家具 |

#### 鲁班锁（孔明锁）
- **原理**：6根或更多木条互相咬合，形成稳定结构
- **种类**：六根、九根、十二根、十八根等
- **应用**：益智玩具，也体现榫卯的精妙
- **制作要点**：精确切割凹槽，每根的槽位不同

### 🏠 传统木工口诀

> **"长木匠，短铁匠，不长不短是石匠"**
> 木工下料宁长勿短，铁器宁短勿长。

> **"凿一凿，退三退"**
> 凿榫眼时要逐步推进，不能急。

> **"刨子要稳，锯子要准，斧子要狠"**
> 三种工具三种用法。

> **"木看纹理，人看脾气"**
> 顺着木材纹理加工才省力。

## 三、冶金铸造

### 🔥 青铜铸造

#### 青铜配方
- **六齐（《考工记》）**：
  - 钟鼎之齐：六分金（铜）一锡
  - 斧斤之齐：五分金一锡
  - 戈戟之齐：四分金一锡
  - 大刃之齐：三分金一锡
  - 削杀矢之齐：五分金二锡
  - 鉴燧之齐：金锡各半

#### 失蜡法
1. 用蜡雕刻模型
2. 外裹耐火泥
3. 加热化蜡流出（留空腔）
4. 浇入铜液
5. 打碎泥模取出铜器
- **代表**：曾侯乙编钟（战国）

#### 现代体验
- 可用石膏+蜡烛做简化版失蜡铸造
- 用锡合金做小件铸件

### ⚔️ 铁器与钢铁

#### 百炼钢
- 反复折叠锻打，去除杂质
- 折叠次数 = "百炼"（实际约30-50次）
- **代表**：干将莫邪、龙泉宝剑

#### 灌钢法（南北朝·綦毋怀文）
- 将生铁液灌入熟铁中，快速渗碳
- 比百炼钢效率高很多

#### 淬火技术
- 加热到红/白色后急速冷却
- 水淬硬但脆，油淬韧但不够硬
- **日本刀**的覆土烧刃借鉴了中国淬火技术

## 四、丝织印染

### 🧵 缫丝与织造

#### 缫丝
- 蚕茧热水煮后抽丝
- 一粒茧可抽出800-1000米丝
- 多根合并成线

#### 织机种类
| 织机 | 特点 | 代表织物 |
|------|------|---------|
| 腰机 | 最原始，绑在腰间 | 葛布、麻布 |
| 斜织机 | 脚踏提综 | 普通丝绸 |
| 花楼织机 | 两人配合，提花 | 蜀锦、宋锦、云锦 |

### 🎨 缂丝
- **"通经断纬"**：纬线不贯穿全幅，按花纹断换
- **特点**：两面花纹相同，如雕琢镂刻
- **代表**：宋代缂丝书画

### 🧶 四大名锦
| 名锦 | 产地 | 特点 |
|------|------|------|
| **云锦** | 南京 | 织金夹银，皇家专用 |
| **蜀锦** | 成都 | 色彩艳丽，图案丰富 |
| **宋锦** | 苏州 | 色彩典雅，格调清新 |
| **壮锦** | 广西 | 对比强烈，几何纹样 |

### 🎭 染色技术

#### 草木染
| 植物 | 颜色 | 用法 |
|------|------|------|
| 蓝草（靛蓝） | 蓝色 | 发酵制靛，还原染色 |
| 茜草 | 红色 | 根部提取，媒染 |
| 栀子 | 黄色 | 果实水煮 |
| 紫草 | 紫色 | 根部酒精提取 |
| 皂斗（橡壳） | 黑色 | 含单宁+铁媒染 |

#### 蜡染
1. 用蜡刀蘸蜡在布上画花纹
2. 浸入染缸（蜡处不上色）
3. 热水煮去蜡
4. 留下白色花纹

#### 扎染
1. 按设计折叠布料
2. 用线绳捆扎
3. 浸染
4. 拆线后形成花纹

## 五、陶瓷工艺

### 🏺 制瓷七十二道工序

#### 核心流程
1. **采石**：取高岭土和瓷石
2. **粉碎**：水碓舂碎
3. **淘洗**：去除杂质
4. **陈腐**：泥料放置数月
5. **拉坯**：在转轮上塑形
6. **利坯**：修整薄厚
7. **画坯**：绘制纹样
8. **施釉**：浸釉/浇釉/吹釉
9. **烧窑**：1280-1350°C高温

#### 各窑特点
| 窑系 | 代表 | 特点 |
|------|------|------|
| **汝窑** | 天青釉 | 釉如堆脂，雨过天青 |
| **官窑** | 粉青釉 | 紫口铁足，冰裂纹 |
| **哥窑** | 金丝铁线 | 开片纹理 |
| **定窑** | 白瓷 | 刻花印花 |
| **钧窑** | 窑变釉 | 入窑一色，出窑万彩 |
| **景德镇** | 青花瓷 | 白地蓝花，天下闻名 |

## 六、其他传统技艺

### 🪨 玉雕
- **"玉不琢不成器"**
- 工序：相玉→设计→琢磨→抛光
- **技法**：圆雕、浮雕、镂雕、俏色

### 🔔 景泰蓝（铜胎掐丝珐琅）
1. 制胎：铜片锤打成型
2. 掰丝：铜丝弯成花纹
3. 焊丝：粘在铜胎上
4. 点蓝：填入珐琅釉
5. 烧蓝：高温烧结（反复3次）
6. 磨光：打磨平整
7. 镀金：露出的铜丝镀金

### 🧻 剪纸
- **北方风格**：粗犷豪放（陕北、山东）
- **南方风格**：细腻精巧（扬州、广东）
- **技法**：阴剪（剪去图案部分）、阳剪（剪去背景部分）

### 🏮 油纸伞
- 骨架：竹子削成伞骨
- 糊纸：涂刷桐油的皮纸
- 绑线：棉线串联伞骨
- **文化**：客家婚嫁必备，"伞"谐"散"但实际寓意人丁兴旺
