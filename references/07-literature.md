## 一、诗词格律

### 📝 诗的基本格式

#### 五言绝句
- 四句，每句五字
- 押韵：第一句可押可不押，二四句必押
- 平仄基本格式（以仄起首句不入韵为例）：
```
仄仄平平仄
平平仄仄平（韵）
平平平仄仄
仄仄仄平平（韵）
```

#### 七言绝句
- 四句，每句七字
- 基本格式（以平起首句入韵为例）：
```
平平仄仄仄平平（韵）
仄仄平平仄仄平（韵）
仄仄平平平仄仄
平平仄仄仄平平（韵）
```

#### 五言律诗 / 七言律诗
- 八句，中间两联（颔联、颈联）必须对仗

#### 对仗要求
- 词性相同（名对名、动对动）
- 平仄相反
- 意义相关或相对
- **经典对仗**：
  - "无边落木萧萧下，不尽长江滚滚来"（杜甫）
  - "两个黄鹂鸣翠柳，一行白鹭上青天"（杜甫）

### 🖊️ 写诗速成

#### 写七绝步骤
1. **定主题**：想写什么（春景、离别、感怀）
2. **选韵脚**：选一个韵部（如"江"韵：江、窗、双、幢）
3. **起句**：交代场景或时间
4. **承句**：承接起句，展开画面
5. **转句**：转折或深化
6. **合句**：收束全诗，点明主旨

## 二、书法

### 五体简介

| 书体 | 特点 | 代表人物 |
|------|------|---------|
| **篆书** | 古朴圆转 | 李斯、邓石如 |
| **隶书** | 蚕头燕尾 | 蔡邕、伊秉绑 |
| **楷书** | 端正规范 | 颜真卿、柳公权、欧阳询、赵孟頫 |
| **行书** | 流畅自然 | 王羲之、苏轼 |
| **草书** | 龙飞凤舞 | 张旭、怀素、王铎 |

### 临帖推荐

| 阶段 | 推荐字帖 | 理由 |
|------|---------|------|
| 入门 | 欧阳询《九成宫》 | 法度严谨，适合打基础 |
| 进阶 | 颜真卿《多宝塔》 | 气势雄浑，结构宽博 |
| 行书 | 王羲之《兰亭序》 | 天下第一行书 |
| 隶书 | 《曹全碑》 | 秀美飘逸 |
| 草书 | 孙过庭《书谱》 | 草书入门最佳 |

### 书法口诀
> **"永字八法"**：侧（点）、勒（横）、弩（竖）、趯（钩）、策（挑）、掠（撇）、啄（短撇）、捺——一个"永"字包含八种基本笔法。

> **"欲左先右，欲下先上"**：起笔方向与行笔方向相反，蓄力发力。

## 三、绘画

### 国画分类

| 类别 | 特点 | 代表 |
|------|------|------|
| **山水** | 意境深远 | 茿宽《溪山行旅图》、黄公望《富春山居图》 |
| **花鸟** | 生动传神 | 齐白石虾、徐悲鸿马 |
| **人物** | 传神写照 | 顾恺之《洛神赋》、吴道子 |

### 技法
- **工笔**：细致精微，三矾九染
- **写意**：笔墨淋漓，以形写神
- **白描**：纯线造型，不施色彩
- **没骨**：不用墨线勾勒，直接用色彩画

### 六法论（谢赫）
1. **气韵生动**：画面要有生命力（最高标准）
2. **骨法用笔**：笔力要有骨气
3. **应物象形**：造型要准确
4. **随类赋彩**：设色要合理
5. **经营位置**：构图要得当
6. **传移模写**：临摹学习

## 五、春联与对联

### 对联基本规则
1. **字数相等**
2. **词性相对**（名对名、动对动）
3. **平仄相反**（上联仄收，下联平收）
4. **意义相关或相对**

### 经典春联

| 上联 | 下联 | 横批 |
|------|------|------|
| 天增岁月人增寿 | 春满乾坤福满门 | 春回大地 |
| 爆竹声中一岁除 | 春风送暖入屠苏 | 新春大吉 |
| 一帆风顺年年好 | 万事如意步步高 | 吉星高照 |
| 绿竹别其三分景 | 红梅正报万家春 | 春回大地 |
| 和顺一门有百福 | 平安二字值千金 | 万象更新 |

### 写春联技巧
1. **定主题**：迎春、祝福、家风
2. **找平仄**：上联末字仄声，下联末字平声
3. **选意象**：春、福、喜、财、平安
4. **对仗**：词性相同，意义呼应

### 常用春联横批
紫气东来、春回大地、万事如意、吉星高照、福寿康宁、五福临门、欣欣向荣、国泰民安
