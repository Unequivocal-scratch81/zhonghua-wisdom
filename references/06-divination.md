### 🎲 简易起卦法

#### 硬币占卦（三币法）
1. 准备三枚硬币，正面=3（阳），反面=2（阴）
2. 同时抛三枚，记录总和
   - 6（三反）= 老阴 ⚋ → 变爻
   - 5（两反一正）= 少阳 ⚊
   - 4（两正一反）= 少阴 ⚋
   - 6（三正）= 老阳 ⚊ → 变爻
3. 抛六次，从下往上画卦
4. 有变爻的要变卦（阴阳互变）

#### 数字起卦法
- 随机取三个数字
- 第一个数÷8取余=上卦
- 第二个数÷8取余=下卦
- 第三个数÷6取余=动爻

## 二、解梦术

### 💤 传统解梦理论

**《周公解梦》体系**：梦是身体状态和心理状态的反映。

#### 梦的分类
1. **正梦**：日常思虑所至
2. **噩梦**：身体不适或心理压力
3. **思梦**：日有所思夜有所梦
4. **寤梦**：半睡半醒时的梦
5. **喜梦**：喜悦之事
6. **惧梦**：恐惧之事

#### 常见梦境解读

| 梦境 | 传统解读 | 心理学视角 |
|------|---------|-----------|
| **梦见蛇** | 吉：蛇为小龙，主财运；凶：蛇缠身有小人 | 蛇代表性、智慧或恐惧 |
| **梦见水** | 清水吉，浊水凶；大水主财 | 水代表情感、潜意识 |
| **梦见火** | 火旺主旺运；火灾主破财 | 火代表激情、愤怒、净化 |
| **梦见牙齿脱落** | 亲人有事；自身健康需注意 | 焦虑、无力感、变老恐惧 |
| **梦见飞** | 高飞主升迁；跌落主挫折 | 渴望自由、掌控力 |
| **梦见考试** | 准备不足的焦虑 | 自我评价、面对挑战 |
| **梦见蛇追** | 有事纠缠不清 | 逃避问题 |
| **梦见已故亲人** | 托梦：需祭祀或有未完成的心愿 | 思念、未了情感 |
| **梦见怀孕** | 新计划、新开始 | 创造力、新生事物 |
| **梦见掉入深渊** | 失控感、恐惧 | 对未知的焦虑 |
| **梦见被追赶** | 有压力在追 | 逃避现实问题 |
| **梦见迷路** | 方向感缺失 | 人生困惑、选择困难 |

#### 梦境组合分析法
- **时间**：子时（23-1）梦主近期，午时梦主远期
- **情境**：梦中情绪比内容更重要
- **反复**：反复出现的梦是潜意识的强烈信号
- **颜色**：彩色梦比黑白梦更强烈

## 三、风水堪舆

### 🏠 阳宅风水（住宅）

#### 基本原则
1. **藏风聚气**：背山面水，气不外泄
2. **明堂开阔**：门前视野宽广
3. **四象俱全**：左青龙、右白虎、前朱雀、后玄武

#### 家居风水要点

**大门**：
- 忌对电梯（气场直冲）
- 忌对楼梯（气往下泄）
- 忌门对门（对冲）
- 宜有玄关（缓冲气场）

**客厅**：
- 宜在房屋前方
- 沙发靠墙（有靠山）
- 电视不宜正对大门

**卧室**：
- 床头靠实墙
- 忌床头对门
- 忌镜子对床
- 忌横梁压顶

**厨房**：
- 灶台不对水槽（水火相冲）
- 灶台不对门（财气外泄）
- 宜在东方或东南方（木生火）

**书房/办公**：
- 座位背后有墙
- 面向门口或窗户
- 左边高右边低（青龙高于白虎）

#### 五行与颜色

| 方位 | 五行 | 适合颜色 | 不利颜色 |
|------|------|---------|---------|
| 东 | 木 | 绿、青 | 白、金 |
| 南 | 火 | 红、紫 | 黑、蓝 |
| 西 | 金 | 白、银 | 红、火 |
| 北 | 水 | 黑、蓝 | 黄、土 |
| 中 | 土 | 黄、棕 | 绿、木 |
